Alias to SSH into dev box
=========================

1. Open `~/.bash_profile`

  ```bash
  subl ~/.bash_profile
  ```

2. Add the following to it
  
  ```bash
  alias devbox="ssh brbennet-ld1" # SSH into dev box faster
  ```

3. Save it and then source `~/.bash_profile` to add your change to the terminal session

  ```bash
  source ~/.bash_profile
  ```
